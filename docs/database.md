# database  for  dummies
## how to deal with our setup in 4 steps 
1. go to neon.com 
2. setup accoutn and create a new project
3. get the connection string and update your .env file with it you will find example in the .env.example file in the root directory
4.  inside the console of neon run 'CREATE EXTENSION IF NOT EXISTS vector;' to enable vector extension
